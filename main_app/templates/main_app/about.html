"FeedMeWell" Project

Originally "FeedMeWell" Project was based on Telegram bot only, but it was decided to make a simple web version with limited functionality. To check out all the features of this project listed below, please use telegram bot.

Location

No further than "X" km from user`s location or an option to choose available city

Price ranges

User can choose from 3 different price categories or include all categories

Restaurant`s rating

No lower than "X" (available options: 3, 3,5, 4, 4.5, 5)

Type of cuisine

Multiple choice (example: French, Spanish, Japanese, Filipino, etc)

Output type

Option to get a random restaurant or all recommendations(limited to 10 in bot)
